您现在可以在 GitHub 主页上展示最能代表您工作的仓库。  
使用 “固定仓库” 列表，您可以固定您贡献的任何公共存储库。  
一旦您选择了最多五个存储库，就可以拖放以按照您喜欢的顺序放置它们。
